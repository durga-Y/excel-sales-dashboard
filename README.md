Sales Dashboard – Excel Project

Project Overview

This project showcases a Sales Analytics Dashboard built using Microsoft Excel.
The dashboard provides clear insights into Total Sales, Profit, Region-wise analysis, and overall business performance.

 Key Features

Interactive Excel dashboard

KPIs: Total Sales, Total Profit, Profit Margin

Region-wise performance breakdown

Monthly sales trends

Clean and easy-to-understand visualizations

 Files Included

Sales_Dashboard.pdf – Summary report

 Tools Used

Microsoft Excel

Pivot Tables & Pivot Charts

Slicers

Data Cleaning Techniques

Insights Highlight

Best performing region: The IT and R&D departments stand out as the strongest performers, generating the highest revenue and profit.

Profit margin observation: The company maintains a stable 27.48% profit margin, meaning it keeps ₹27.48 as profit for every ₹100 earned, showing healthy financial efficiency.

Sales performance: Sales and revenue vary noticeably across departments and quarters, with Q2 showing the strongest growth while other periods fluctuate.
